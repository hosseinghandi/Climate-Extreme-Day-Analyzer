## Climate Extreme Day Analyzer 🌡️
A tool for analyzing EPW climate data and identifying the hottest day of the year.

## 🧩 Overview
This component processes EPW weather data and calculates UTCI to detect extreme heat conditions and generate a simulation-ready period for Ladybug environmental analysis.

## 🎯 Project Goals
The goal of this tool was to consolidate the steps required to identify the hottest day into a single automated workflow. It brings together all relevant climate data in one place, simplifying the process of defining and running simulation scenarios for extreme heat analysis.

## 📂📘 Getting Started
To get started, open the provided example files and follow the included instructions.

## ✨ Main Features
- Automatic hottest day detection based on UTCI calculation
- Generation of simulation-ready analysis period (day and hourly range)
- Extraction of key climate parameters (temperature, humidity, wind speed)
  
## ⚠️ Challenges I Faced
- Handling and structuring data required for UTCI calculation
- Ensuring accurate identification of extreme conditions from climate datasets
